Что такое Linux и для чего он применяется.
    Это операционная система, которая распростроняется на бесплатной основе и используется обычно на
    серверах, также на смартфонах (android) и ПК/ноутбуках. Поскольку Linux имеет открытые исходники,
    его можно настроить на свой вкус. Так же эта система является безопаснойб потому что под нее нет вирусов,
    благодаря ее архитиктуре, открытости, большой комиюнити и тп

Чем отличается CLI от GUI.
    CLI - Command Line Interface - ввод комманд из Терминала, необходимо помнить комманды, поэтому сложнее,
    чем GUI. Быстрее чем GUI, потому что коммадны вводятся с клавиатуры из терминала. Кроме того может давать
    больше возможностей, чем предусмотрены GUI
    GUI - Graphical User Interface - исспользуется копютерная графика, можно нажать на кнопку или перетащить
    значек мышкой и тд. Более интуивный интерфейс, легче для новичков

Что такое SSH.
   SSH - Secure Shell Protocol - криптографический сетевой протокол, позволяющий безопасно управлять удаленной
   операционной системой с помощью коммандной строки и передавать файлы

Как сгенерировать пару RSA-ключей и зачем это делать.
    С помощью комманды ssh-keygen -t rsa в Терминале
    RSA-ключи неодходимы для беспарольного доступа к SSH. Приватный ключ используется для шифрования и
    публичный код для расшифровки

каково назначение команды df ?
    df - disk free - комманда для просмотра статистики по дискам
как посмотреть публичный ключ на своем компьютере?
    cat ~/.ssh/id_rsa.pub

Какие задачи решаются с помощью команды 'gzip'.
    Эта комманда используется для компрессиии файлов, то есть уменьшить размер файлов и освободить просторанство
    на диске
